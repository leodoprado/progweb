# Condomínio Web 🏘️

### Aplicação desenvolvida com `EJS + NODE JS + MySQL`

***Instale as dependências do projeto utilizando o comando: npm install***

**Dependências do projeto:**
  * Express
  * Express-session
  * Mysql2
  * Sequelize
  * BcryptJS
  * EJS
  * Body-parser
  * Module-alias
  * Dotenv
  * Connect-flash
